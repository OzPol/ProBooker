## ProBooker

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The homepage is located in /app/page.txt and currently shows a header and form for a consumer user to log in.
